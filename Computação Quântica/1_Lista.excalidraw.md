## Text Elements
a) Para todo algoritmo escrito em C, existe uma Máquina de Turing equivalente
que resolve o mesmo problema, e vice-versa. A razão disso é que, embora estes
dois formalismos sejam diferentes, ambos são capazes de expressar algoritmos
que decidem qualquer linguagem recursiva.
Verdadeiro ou Falso?

Verdadeiro.
Essa afirmação está fundamentada em um dos pilares da Teoria da Computação.
1. A Tese de Church-Turing
    Postula que qualquer processo que possa ser chamado de "algoritmo" ou
    "computação efetiva" pode ser realizado por uma Máquina de Turing. Como a
    linguagem C é um modelo de computação de propósito geral (imperativo), ela
    se enquadra nessa tese.
2. Equivalência de Formalismos
    Embora uma Máquina de Turing opere de forma muito rudimentar e o C utilize
    abstrações de alto nível ambos são Turing-completos.
    C -> MT
        É possível construir uma MT que simule a memória RAM e a execução de
        instruções de um programa em C
    MT -> C
        É trivial escrever um programa em C que simule o comportamento de uma
        fita, a tabela de transições e a cabeça de leitura/escrita de uma MT
3. Linguagens Recursivas e Decidibilidade
    A questão menciona linguagens recursivas, que são aquelas para as quais
    existe uma MT que sempre para (aceitando ou rejeitando a entrada).
    - Se um problema é resolvido por um algoritmo em C que sempre termina, ele
      define uma linguagem recursiva
    - Se uma linguagem é recursiva, existe uma MT que a decide e, consequentemente,
      um programa em C equivalente. ^Hr1YtZdw

 b) Máquinas de Turing são modelos matemáticos formais para representar o que
chamamos de algoritmos. No caso específico da Máquina de Turing Universal (que
é a especificação de uma Máquina de Turing em particular) temos um modelo
matemático para representar o que chamamos de um computador de propósito geral.
Verdadeiro ou Falso? 

Verdadeiro.
Toca no ponto central que permitiu a transição da matemática teórica para a
criação dos computadores modernos.
1. Máquina de Turing
    Uma Máquina de Turing comum é projetada para realizar uma tarefa específica
    (por exemplo, uma máquina que apenas soma dois números ou uma que verifica
    se uma palavra é um palíndromo). Nesse sentido, ela é o modelo formal de um
    algoritmo.
2. Máquina de Turing Universal
    A grande inovação de Alan Turing foi o conceito de MTU. Em vez de ter uma
    máquina fixa para cada tarefa, a MTU recebe duas entradas em sua fita:
    1) A descrição de uma outra MT qualquer (M)
    2) Os dados de entrada (w) para essa máquina.
    A MTU então simula o comportamento de M sobre w. Isso é exatamente o que
    um computador de propósito geral faz: ele não é fixo para uma única função;
    ele lê um programa (software) da memória e executa as instruções contidas nele
    sobre determinados dados.        ^E2daSIoo

c) Autômatos com pilha (APs) são formalismos matemáticos usados para descrever
certos algoritmos, embora nem todo algoritmo possível pode ser expresso em termos
de APs. A razão disso é que o conjunto das linguagens livres de contexto (formalismo
usado para definir problemas decidíveis por APs) está estritamente contido no
conjunto de linguagens recursivas (formalismo para definir todo problema decidível
algoritmicamente, ou seja, todo problema decidível por alguma Máquina de Turing). Em
Teoria da Computação diferentes modelos de computação estão associados a
diferentes conjuntos de problemas, sendo que tipicamente modelos mais gerais (e.g.,
Máquinas de Turing) são capazes de resolver problemas que modelos mais simples (e.g.,
APs, autômatos finitos determinísticos, etc) não são capazes de resolver.
Verdadeiro ou Falso? ^287QXBeg
